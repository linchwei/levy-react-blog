## 首页苹果风格动画改造计划

### 阶段一：HeroSection 苹果风格改造
1. **文字逐行显现动画**
   - 标题每个字依次出现，带弹性效果
   - 使用 Framer Motion 的 staggerChildren
   - 添加文字模糊到清晰的过渡

2. **视差滚动增强**
   - 背景网格以 0.5x 速度滚动
   - 内容以 1x 速度滚动
   - 装饰元素以 1.2x 速度滚动

3. **产品展示式头像动画**
   - 头像从下方滑入 + 缩放
   - 添加阴影深度变化
   - 悬停时的 3D 倾斜效果

4. **状态指示器脉冲动画优化**
   - 更柔和的脉冲效果
   - 多层扩散圆圈

### 阶段二：滚动触发动画系统
1. **创建 ScrollReveal 组件**
   - 通用滚动显现包装器
   - 支持多种动画方向（上、下、左、右、缩放）
   - 可配置延迟和持续时间

2. **Section 分隔动画**
   - 每个 section 进入时标题动画
   - 内容依次 stagger 出现
   - 分割线动画

### 阶段三：BentoGrid 苹果风格改造
1. **卡片悬停 3D 效果**
   - 使用 CSS transform-style: preserve-3d
   - 鼠标位置追踪倾斜
   - 光泽扫过效果

2. **图标动画**
   - 图标从 0 缩放到 1
   - 带弹性回弹效果
   - 渐变色流动动画

3. **数字滚动计数器**
   - 数字从 0 滚动到目标值
   - 使用 requestAnimationFrame
   - 带缓动函数

### 阶段四：FeaturesSection 改造
1. **大标题文字效果**
   - 渐变文字颜色流动
   - 文字阴影深度变化
   - 逐词显现动画

2. **特性卡片序列动画**
   - 卡片依次从下方滑入
   - 每张卡片有递增延迟
   - 图标先出现，文字后出现

### 阶段五：全局优化
1. **导航栏滚动效果**
   - 滚动后背景模糊增强
   - 高度收缩动画
   - Logo 缩放效果

2. **平滑滚动**
   - 使用 Lenis 或类似库
   - 惯性滚动效果
   - 锚点跳转平滑过渡

3. **性能优化**
   - 使用 will-change 属性
   - 图片懒加载
   - 动画 GPU 加速
   - 减少重排重绘

4. **可访问性**
   - 支持 prefers-reduced-motion
   - 键盘导航支持
   - 屏幕阅读器友好

### 技术实现
- **Framer Motion** - 主要动画库
- **GSAP + ScrollTrigger** - 复杂滚动动画
- **CSS 3D Transforms** - 3D 效果
- **Intersection Observer** - 滚动触发检测

### 预期效果
- 首页加载时有电影般的开场动画
- 滚动时每个 section 都有精心设计的入场效果
- 所有交互都有流畅的反馈动画
- 整体体验接近苹果官网的精致感
