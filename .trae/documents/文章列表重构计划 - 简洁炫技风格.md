## 问题分析

当前文章列表采用传统卡片网格布局，显得过于常规，缺乏技术感和视觉冲击力。

## 重构方案

### 设计理念
**极简列表 + 技能炫技 + 视觉焦点**

放弃卡片式网格，采用**杂志风格的极简列表布局**，突出技术感和专业度。

### 具体改动

#### 1. 布局重构 - 极简列表式
- **左侧**: 文章序号（大号数字，视觉焦点）
- **中间**: 标题 + 标签（简洁有力）
- **右侧**: 日期 + 阅读时间（次要信息）
- **悬停效果**: 整行高亮 + 箭头滑入

#### 2. 视觉风格 - 技术感
- **背景**: 深色渐变 + 代码雨/粒子效果（可选）
- **分隔线**: 动态渐变线条，悬停时发光
- **序号**: 大号等宽字体，体现程序员身份
- **标签**: 代码风格标签 `<React />` `<TypeScript />`

#### 3. 交互动画 - 炫技
- **入场**: 列表项依次滑入，带 stagger 延迟
- **悬停**: 
  - 整行背景渐变高亮
  - 箭头从右侧滑入
  - 序号颜色变化
  - 分隔线发光动画
- **滚动**: 视差效果

#### 4. 精选文章突出
- 第一篇或热门文章采用**大标题 + 背景图**的 hero 样式
- 其他文章保持极简列表

### 技术实现
- 使用 Framer Motion 实现流畅动画
- Tailwind CSS 实现渐变和发光效果
- 等宽字体突出技术感
- 玻璃拟态效果增加层次

### 预期效果
- 更简洁、更专业
- 突出技术博主身份
- 视觉上有冲击力但不杂乱
- 交互流畅有惊喜感
