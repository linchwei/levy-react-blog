## UI 优化计划

### 问题分析

1. **Title 顶到导航栏**
   - Hero 区域的顶部没有足够的 padding
   - 需要增加 `pt-24` 或更大的顶部间距

2. **缺少苹果电脑标识**
   - 技术栈卡片需要添加 macOS 风格的窗口装饰
   - 添加窗口控制按钮（红黄绿三个圆点）
   - 添加类似 macOS 的顶部栏设计

3. **页面太过素了**
   - 增加渐变背景和光晕效果
   - 添加更多玻璃拟态（glassmorphism）卡片
   - 增加微妙的阴影和边框发光效果
   - 添加悬浮粒子或装饰元素
   - 优化颜色层次

### 具体改动

#### HeroSection.tsx
- 增加顶部 padding (`pt-24` → `pt-32`)
- 技术栈卡片改为 macOS 风格窗口设计
- 添加窗口控制按钮（红/黄/绿）
- 添加 macOS 风格顶部栏
- 增加背景渐变光晕
- 添加装饰性悬浮元素

#### ProfileCard.tsx
- 优化卡片阴影和边框
- 添加更多渐变装饰
- 增加头像周围的发光效果

#### SkillsSection.tsx
- 优化卡片视觉效果
- 增加渐变边框和阴影
- 添加图标背景动画

### 预期效果
- 页面有足够的呼吸空间，不压抑
- 技术栈区域呈现 macOS 窗口风格
- 整体视觉更丰富、更有层次感
