## 问题分析

当前文章列表页面采用传统的三列卡片网格布局，显得过于常规，缺乏技术感和独特性。

## 重构方案

### 设计理念
**极简列表 + 技能炫技 + 视觉焦点**

放弃卡片网格，采用**类似代码编辑器/终端的列表布局**，突出技术博主的专业身份。

### 具体改动

#### 1. 布局重构 - 编辑器风格列表
- **左侧**: 行号（大号等宽字体，类似代码编辑器）
- **中间**: 文章标题 + 代码风格标签
- **右侧**: 元信息（日期、阅读时间、浏览量）
- **悬停**: 整行高亮 + 行号变色 + 光标效果

#### 2. 搜索筛选区 - 终端风格
- 搜索框改为终端输入框样式
- 分类筛选改为代码注释风格的标签
- 添加 `// Filter by category` 样式的提示

#### 3. 视觉风格 - 代码编辑器
- **字体**: 标题和正文使用等宽字体
- **配色**: 深色主题，语法高亮风格
- **行号**: 灰色等宽数字，悬停时变亮
- **光标**: 悬停时出现闪烁光标效果
- **分隔线**: 代码行号分隔线风格

#### 4. 交互动画 - 炫技
- **入场**: 列表项像代码一样逐行"打印"出现
- **悬停**: 
  - 行号变亮（类似选中行）
  - 整行背景高亮
  - 右侧显示 `>` 箭头
  - 光标闪烁效果
- **滚动**: 平滑滚动 + 视差

#### 5. 代码风格元素
- 标签使用 `// tag` 注释风格
- 日期格式改为 `YYYY-MM-DD`
- 添加 `/* 共 N 篇文章 */` 样式的统计
- 搜索提示用 `# 搜索文章...` 注释风格

### 技术实现
- Framer Motion 实现"打字机"入场效果
- Tailwind CSS 实现编辑器风格配色
- 等宽字体（JetBrains Mono / Fira Code）
- 光标闪烁 CSS 动画

### 预期效果
- 像在看代码编辑器一样浏览文章
- 极简但充满技术感
- 突出程序员身份
- 交互流畅有惊喜
