## 待办事项功能复刻计划

### 目标
将 Vue3 版本的待办事项功能按照 React 版本 1:1 复刻，包括所有动画效果、拖拽排序、统计卡片等功能。

### 详细任务清单

#### 1. 更新 Todo Store

**stores/todo.ts 更新：**
- [ ] 添加 `priorityStats` getter（高/中/低优先级统计）
- [ ] 添加 `reorderTodos` action（拖拽排序）
- [ ] 添加 `clearCompleted` action（清除已完成）
- [ ] 确保数据持久化（pinia-plugin-persistedstate）

#### 2. 创建待办事项页面 (TodoView.vue)

**Hero Section：**
- [ ] 浮动粒子背景动画
- [ ] 动态网格背景
- [ ] 滚动视差效果
- [ ] 标题、描述、特性图标

**主要内容区域：**
- [ ] TodoStats 统计卡片
- [ ] AddTodo 添加表单
- [ ] TodoList 任务列表

#### 3. 创建组件

**TodoStats.vue**
- [ ] 4个统计卡片（总任务/进行中/已完成/完成率）
- [ ] 数字增长动画
- [ ] 3D 倾斜效果
- [ ] 背景渐变动画
- [ ] 装饰元素旋转动画

**AddTodo.vue**
- [ ] 表单验证（zod + vee-validate）
- [ ] 展开/收起动画
- [ ] 优先级选择器
- [ ] 提交成功彩带效果
- [ ] 渐变背景动画

**TodoList.vue**
- [ ] 搜索功能
- [ ] 筛选标签（全部/进行中/已完成）
- [ ] 优先级筛选按钮
- [ ] 拖拽排序（@dnd-kit/vue）
- [ ] 空状态动画
- [ ] 列表进入动画

**TodoItem.vue**
- [ ] 完成勾选动画
- [ ] 完成时彩带效果
- [ ] 优先级标签
- [ ] 删除按钮（悬停显示）
- [ ] 拖拽手柄
- [ ] 进度条动画
- [ ] 背景光效
