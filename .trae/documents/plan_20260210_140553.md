# 时间线页面 UI 重构计划（苹果官网风格参考）

## 一、设计分析

### 苹果官网设计特点
1. **极简主义**：干净、整洁的布局，大量留白
2. **排版**：清晰、优雅的字体层次结构
3. **视觉层次**：强烈的视觉层次感和内容组织
4. **微交互**：微妙、有目的的动画和过渡效果
5. **色彩**：有限的色彩 palette，战略性使用强调色
6. **布局**：基于网格的布局，强对齐
7. **响应式**：无缝的响应式设计
8. **质感**：精致的视觉细节和质感

### 当前页面问题
1. **设计杂乱**：布局拥挤，缺乏苹果设计的极简主义
2. **排版**：字体层次结构不够清晰
3. **色彩**：色彩使用不一致，缺乏和谐感
4. **动画**：动画效果不够微妙和有目的
5. **布局**：缺乏基于网格的结构化布局
6. **质感**：缺乏精致的视觉细节和质感
7. **响应式**：响应式设计可以进一步优化

## 二、重构目标

### 设计目标
1. **极简优雅**：采用苹果风格的极简主义设计
2. **视觉层次**：建立清晰的视觉层次结构
3. **精致细节**：添加微妙的视觉细节和质感
4. **流畅动画**：实现苹果风格的流畅微交互
5. **色彩和谐**：使用有限且和谐的色彩 palette
6. **排版精美**：优化字体层次和排版

### 功能目标
1. **保持核心功能**：筛选、搜索、详情展开等功能
2. **提升用户体验**：更直观、流畅的用户体验
3. **响应式优化**：完美适配所有设备尺寸
4. **性能优化**：保持页面加载速度和交互流畅度

## 三、具体实现方案

### 1. 整体布局重构
- **英雄区**：采用苹果风格的全屏英雄区，简洁的标题和副标题
- **控制面板**：集成到顶部，更简洁的搜索和筛选设计
- **时间线布局**：采用更干净的垂直时间线设计
- **职业发展轨迹**：重新设计为更符合苹果风格的数据可视化
- **页脚**：保持简洁的页脚设计

### 2. 视觉设计升级
- **色彩方案**：
  - 主色调：白色/深空灰背景
  - 强调色：苹果风格的蓝色
  - 中性色：不同灰度层次
- **排版**：
  - 使用无衬线字体，优化字体大小层次
  - 清晰的标题和正文区分
  - 恰当的行高和字间距
- **卡片设计**：
  - 更薄的边框或无边框设计
  - 微妙的阴影效果
  - 干净的圆角设计
- **图标**：
  - 使用线性图标，保持风格统一
  - 适当的图标大小和间距

### 3. 动画效果优化
- **滚动动画**：更微妙的滚动触发动画
- **过渡效果**：平滑的页面元素过渡
- **交互反馈**：微妙的悬停效果和点击反馈
- **加载动画**：简洁的加载指示器

### 4. 功能组件重构
- **搜索框**：更简洁的搜索框设计
- **筛选器**：更优雅的筛选按钮设计
- **时间线卡片**：重新设计的时间线事件卡片
- **职业发展轨迹**：更符合苹果风格的时间线可视化
- **响应式设计**：优化移动端和桌面端布局

### 5. 技术实现
- **样式**：使用 Tailwind CSS 实现苹果风格的设计
- **动画**：使用 Framer Motion 实现流畅的微交互
- **组件**：重构现有组件，保持功能的同时提升 UI
- **性能**：优化动画性能，确保流畅的用户体验
