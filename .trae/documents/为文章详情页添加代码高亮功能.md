## 计划概述
为博客文章详情页添加代码片段语法高亮功能，提升技术文章的可读性。

## 实施步骤

### 1. 安装依赖
- 安装 `react-syntax-highlighter` 用于代码高亮
- 安装 `@types/react-syntax-highlighter` 类型定义

### 2. 更新 BlogDetailPage 组件
- 导入 SyntaxHighlighter 组件
- 在 ReactMarkdown 的 components 配置中添加 code 渲染器
- 根据代码块的语言属性应用对应的语法高亮
- 支持深色/浅色主题自适应

### 3. 添加代码复制功能
- 为代码块添加复制按钮
- 点击后复制代码到剪贴板
- 显示复制成功提示

### 4. 样式优化
- 代码块圆角、内边距
- 行号显示（可选）
- 语言标签显示
- 与整体暗色/亮色主题协调

## 技术方案
- 使用 `react-syntax-highlighter` 的 Prism 渲染器
- 深色主题使用 `vscDarkPlus` 或 `atomDark`
- 浅色主题使用 `prism` 或 `github`
- 通过 CSS 变量或状态管理切换主题
