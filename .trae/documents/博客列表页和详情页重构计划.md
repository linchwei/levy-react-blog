## 博客列表页和详情页重构计划

### 目标
将 Vue3 版本的博客列表页和详情页按照 React 版本 1:1 复刻，包括所有动画效果、AI 功能和交互。

### 详细任务清单

#### 1. 博客列表页 (BlogListView.vue)

**功能特性：**
- [ ] 页面头部动画（标题、描述）
- [ ] 搜索框（带图标）
- [ ] 分类筛选按钮组（全部、Frontend、Backend、Design、DevOps）
- [ ] 文章卡片网格布局
  - 封面图片（悬停缩放）
  - 分类标签
  - 标题（悬停变色）
  - 摘要
  - 标签列表
  - 日期、阅读时间、浏览量、点赞数
- [ ] 空状态提示
- [ ] 进入动画（交错延迟）

**Store 更新：**
- [ ] 添加 `getFilteredPosts` getter
- [ ] 确保筛选逻辑正确

#### 2. 博客详情页 (BlogDetailView.vue)

**Hero Section：**
- [ ] 背景图片（带遮罩）
- [ ] 返回按钮动画
- [ ] 分类标签
- [ ] 文章标题动画
- [ ] 文章摘要
- [ ] 元信息（日期、阅读时间、浏览量、点赞数）

**内容区域：**
- [ ] Markdown 渲染
- [ ] 代码高亮（带行号、复制按钮）
- [ ] 标签列表（可点击跳转）
- [ ] 喜欢按钮
- [ ] 分享按钮

**侧边栏：**
- [ ] AI 文章总结组件
- [ ] 作者卡片
- [ ] 相关文章列表
- [ ] 文章分类导航
- [ ] 工具箱下拉菜单

**AI 组件：**
- [ ] ArticleSummary.vue - 文章总结卡片
- [ ] AIChatWidget.vue - AI 聊天悬浮窗
- [ ] ChatMessage.vue - 聊天消息组件

#### 3. AI 服务层

**composables/useAIChat.ts**
- [ ] AI 聊天逻辑
- [ ] 消息管理
- [ ] 发送消息
- [ ] 重新生成
- [ ] 清空对话

**composables/useArticleSummary.ts**
- [ ] 文章总结生成
- [ ] 加载状态
- [ ] 错误处理

**services/aiService.ts**
- [ ] DeepSeek API 集成

#### 4. Markdown 渲染

**依赖安装：**

**代码高亮组件：**
