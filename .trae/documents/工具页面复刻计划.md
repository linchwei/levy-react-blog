## 工具页面复刻计划

### 目标
将 Vue3 版本的工具页面按照 React 版本 1:1 复刻，包括所有工具分类、动画效果和交互功能。

### 差异对比

**React 版本特性：**
- 渐变标题动画
- 10个工具分类卡片
- 分类图标（渐变背景）
- 工具列表（名称+描述）
- 内部工具链接（ArrowRight 图标）
- 外部工具链接（ExternalLink 图标）
- 悬停显示箭头按钮
- 交错进入动画

**Vue 版本当前问题：**
- 只有4个工具卡片
- 布局不同
- 缺少动画效果
- 没有分类概念

### 详细任务清单

#### 1. 重写 ToolsView 页面

**views/ToolsView.vue**
- [ ] 渐变标题（紫→蓝）

#### 2. 工具分类数据

定义10个分类：
1. 开发工具（VS Code, WebStorm, Cursor）
2. CSS 工具箱（CSS动画库, AI动画生成器, SVG动画编辑器, 滚动动画设计器）
3. 代码工具（代码游乐场）
4. 性能工具（性能分析仪表盘）
5. 3D 工具（3D场景构建器）
6. 设计工具（Figma, Sketch, Framer）
7. 终端工具（iTerm2, Warp, Oh My Zsh）
8. 浏览器（Chrome, Arc, Firefox）
9. 数据库（TablePlus, MongoDB Compass, Redis Insight）
10. 生产力（Notion, Raycast, Alfred）
