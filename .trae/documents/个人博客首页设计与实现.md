# 个人博客首页设计方案

## 设计愿景
打造一个现代、炫酷的个人博客首页，融合待办事项功能，采用深色主题配合玻璃拟态效果，营造科技感和创意氛围。

## 核心功能模块

### 1. 炫酷Hero区域
- 动态粒子背景动画
- 个人头像与简介
- 打字机效果的标语
- 社交媒体链接
- 滚动指示器动画

### 2. 导航栏
- 固定顶部导航
- 玻璃拟态效果
- 平滑滚动到各区域
- 移动端汉堡菜单
- 深色/浅色模式切换

### 3. 功能入口卡片 (Bento Grid布局)
- **待办事项入口** - 保留现有功能，显示待办统计
- **文章列表入口** - 最新文章预览
- **项目展示入口** - 个人项目作品集
- **关于我入口** - 个人简介和技能
- **工具箱入口** - 常用工具推荐
- **时间线入口** - 个人经历时间线

### 4. 最新文章区域
- 文章卡片网格布局
- 分类标签
- 阅读时间估算
- 悬停动画效果

### 5. 技能展示
- 技能标签云
- 进度条动画
- 图标展示

### 6. 页脚
- 版权信息
- 快速链接
- 订阅 Newsletter
- 社交链接

## 技术实现

### 新增依赖
- framer-motion - 动画库
- react-particles - 粒子背景
- @react-spring/web - 弹簧动画

### 路由调整
```
/                    -> HomePage (新首页)
/blog                -> BlogListPage (文章列表)
/blog/:slug          -> BlogDetailPage (文章详情)
/todo                -> TodoPage (待办事项 - 保留)
/todo/:id            -> TodoDetailPage (待办详情 - 保留)
/projects            -> ProjectsPage (项目展示)
/about               -> AboutPage (关于我)
/tools               -> ToolsPage (工具箱)
/timeline            -> TimelinePage (时间线)
```

### 设计特色
1. **玻璃拟态效果** - 半透明背景配合模糊效果
2. **渐变色彩** - 紫色到蓝色的渐变主题
3. **微交互动画** - 悬停、点击的细腻动画
4. **粒子背景** - Hero区域的动态粒子效果
5. **响应式设计** - 完美适配移动端和桌面端
6. **深色主题** - 默认深色模式，支持切换

### 动画效果
- 页面加载动画序列
- 滚动触发动画
- 卡片悬停3D效果
- 打字机效果
- 数字滚动动画
