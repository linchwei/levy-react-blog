## 当前问题分析

当前项目展示页面采用传统卡片网格布局，过于简单，缺乏设计感、科技感和码农特色。

## 重构方案

### 设计理念
**科技感 + 码农风格 + 视觉冲击力**

采用类似 GitHub/VS Code 的界面风格，结合终端元素和代码高亮，打造独特的项目展示体验。

### 具体改动

#### 1. 页面头部 - 终端风格
- 终端窗口样式标题区
- 命令行风格介绍文字
- 动态打字机效果
- 代码语法高亮装饰

#### 2. 项目卡片 - 代码编辑器风格
- **左侧**: 行号 + 文件图标
- **中间**: 项目名称（代码高亮）+ 描述
- **右侧**: 技术栈标签（彩色徽章）
- **底部**: GitHub stars/forks 统计 + 操作按钮
- **悬停**: 代码选中效果 + 光标闪烁

#### 3. 项目特色展示
为不同类型项目设计不同吸引点：
- **AI 项目**: 神经网络动画背景 + 模型指标展示
- **前端项目**: 组件预览图 + 性能评分
- **全栈项目**: 架构图 + API 端点列表
- **工具项目**: 终端演示动画 + 使用统计

#### 4. 视觉元素
- **背景**: 渐变光晕 + 代码雨效果
- **分隔线**: 动态渐变线条
- **标签**: 彩色徽章 + 悬停发光
- **按钮**: 终端风格按钮

#### 5. 交互动画
- 卡片悬停 3D 倾斜效果
- 标签悬停放大 + 发光
- 数字递增动画
- 滚动视差效果

#### 6. 代码风格元素
- 等宽字体贯穿全页
- 代码注释风格说明
- 语法高亮配色
- 终端光标效果

### 技术实现
- Framer Motion 实现复杂动画
- Tailwind CSS 实现玻璃拟态和渐变
- 等宽字体（JetBrains Mono）
- 响应式设计

### 自测要求
- TypeScript 无错误
- ESLint 无错误
- 浏览器控制台无错误
- 主题切换正常
